# ws

[![Codecov](https://img.shields.io/codecov/c/github/nhooyr/ws.svg)](https://codecov.io/gh/nhooyr/ws)

ws is a clean and idiomatic WebSocket library for Go.

This library is in heavy development.

## Install

```bash
go get nhooyr.io/ws
```
